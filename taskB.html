Nápověda pro téma ATRIBUTY = vlastnosti

Obrázek má tag img, jedná se o nepárový tag, takže za ním nemusí být patka </img >. Jeho povinný atribut je src="...", kde se nachází odkaz na obrázek. Pro formátování doporučujeme použít atribut style, kam vkládáme tzv. CSS formátovací příkazy. Pro šířku je to příkaz width, pro zarovnání float a pro odsazení margin.

Odkaz = a TAG Obrázek = img TAG Formulářový prvek = input TAG
